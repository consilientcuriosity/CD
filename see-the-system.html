See the system

Flow meets boundaries. If recovery is missing, stress accumulates.

CD uses a simple visual language to help people notice capacity loss early. The goal is not drama — it’s diagnosis.

How to read CD visuals

What you seeWhat it means
Crack locations stay fixedHistory matters. Damage is path-dependent.
Cracks widen slowlyRecovery is failing. Capacity is being used up.
Patchiness increasesFragmentation replaces coherence.
No explosions or shatterCollapse is not a “moment”; it’s an accumulation.
CD visuals do not predict outcomes. They support reflection: where is load exceeding recovery? Where are limits being ignored?

Interactive: clinical depletion

Clinical depletion: the system continues to “work,” but its ability to recover or adapt is declining.

Animation script (public)

  • Story: healthy → warning window → depletion → brittle state
  • Invariant: crack locations do not move
  • Primary signal: widening and thinning
  • Pacing: slow (12–24 seconds if auto-played)
CD focuses on capacity, limits, and recovery — not blame, optimisation, or spectacle.
